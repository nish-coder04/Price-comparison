# 🛒 QuickCompare — Price Comparison Tool

A web app that compares grocery prices across quick-commerce platforms — **Blinkit, Zepto, and Instamart** — to help you find the cheapest deal.

---

## 🛠️ Tech Stack
- Python (Flask)
- SQLite (database)
- Selenium (web scraping)

---

## ⚙️ Features
- Search a product and compare live prices across Blinkit, Zepto, and Instamart
- Stores price history in a local database
- Login/session management
